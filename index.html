Pizza Shop Server

The REST API for the pizza ordering application. Built with Node.js and Express.
*This server is using lowdb as a database and save the new data to a .json file, so new data won't persist.*
Source Code | iOS application

Method Endpoint Description
GET /api/v1/foods Get all foods
GET /api/v1/orders Get all orders
POST /api/v1/orders/add Add a new order
DELETE /api/v1/orders/delete/:id Delete order by id
GET /api/v1/reservations Get all reservations
POST /api/v1/reservations/add Add a new reservation
DELETE /api/v1/reservations/delete/:id Delete reservation by id
GET /api/v1/users Get all users
GET /api/v1/users/login Login
POST /api/v1/users/register Register
DELETE /api/v1/users/delete/:phone Delete user by phone number
